As I told you earlier, when we want to send an event or value from a child component to the parent component, we use $emit

If we want to pass data and values from parent component to child component, we use props to Passes array values (notes) from the parent component App.Vue to the child component ListNotes.vue

We have to write the v-bind directive as follows
<template>
   <CreateNote v-on:SeDataToCnote="CreateNote"/>
   <ListNote v-bind:notesd="notes"/>  ✔️
</template>

As you can see, we wrote the notesd name next to v- bind
v-bind: notesd = "notes"
So we invoked the notes array, as a value for the notesd name as you can see
v-bind: notesd = "notes"


You may be wondering what is the point of v-bind: notesd = "notes"
The benefit of this is that we send the values of the notes array to the ListNote child component.
The values are received there in ListNotes.vue using props
