    use Illuminate\Support\Facades\Config;
use Illuminate\Support\Facades\Session;
use Illuminate\Support\Facades\Redirect;
 public function switchLanguage($lang)
  {
       if (array_key_exists($lang,Config::get('languages')))){
       Session::put('c_applocale',$lang);
       } return Redirect::back();
    }

    This function checks the first two characters of the language name, entered in the path,
    Route:: get('lang / {lang}',
    If it's not there, in the languages.php file, it won't do anything, it will just return

    If it is located in languages.php file. These two literals of the language name will be stored in the session,
    And it's okay, you can translate the text of the home.blade.php file page to the language specified by the path

    Now that the page is reloaded, we want to set the language we chose earlier
    We must create Middleware


    php artisan make:middleware Language.php
    After creating /app/Http/Middleware/Language.php/
     We put this code inside a function
    handle

     

        public function handle(Request $request, Closure $next){  
       if (Session()->has('c_applocale') AND array_key_exists(Session()->   get('c_applocale'),config('languages'))) {
          App::setLocale(Session()->get('c_applocale')); 
        }
        return $next($request);
    }

    At the beginning of entering the web page, This The code 
    App::setLocale(Session()->get('c_applocale'));
