Clarification and explanation of the maintenance mode in laravel

2022.12.07 by Murad Shaker

When you want to make some changes or make maintenance in your laravel project that is used by a number of people, in this case you should show users a message that the site is under maintenance

Fortunately, Laravel makes it very easy to put your app into maintenance mode!

In this tutorial, I'll show you how to do that, as well as how to change the default Laravel maintenance page with a custom maintenance page that matches your website's design!

 

 

 

 

---------------------------

Note::: When you want to update or change your project codes, you may be afraid that users will send data to the database or the like.
In this case, it is better to disable the laravel project temporarily until the maintenance is done
This is done using the maintenance mode found in laravel

Create Custom Maintenance Page

Navigate to resources/views folder And create a folder named errors. Under errors folder, create a file named 503.blade.php   paste your design
resources\views\errors\503.blade.php 



  
  
      


 

  

    
503

  

    
We are currently down for maintenance

  
We will be up in couple of hours. Thanks for patience

 

Enable Default Maintenance Mode

Now when you type the php artisan down in the Command Prompt,
 php artisan down 
Maintenance mode will be enabled, and when the laravel webpage is reloaded, Laravel will immediately look for the 503.blade.php file, and display the content of the html elements, which are in the 503.blade.php file, On your project webpage

As mentioned above, there is a message in the 503.blade.php file that alerts the user
The site is under maintenance if you start maintenance mode,
When the user enters the private website, the alert page will appear immediately

Now let's make sure
Type this in the command prompt
php artisan serve
Then paste this http://127.0.0.1:8000
in your browser URL and click Enter

Now you may notice a user alert interface warning that the site is under maintenance
as you see

 

Disable the default maintenance mode
When you have completed the maintenance, you want to return the display of project content in the web to normal
If you want to disable maintenance mode
Type the following command in the command prompt
 php artisan up
